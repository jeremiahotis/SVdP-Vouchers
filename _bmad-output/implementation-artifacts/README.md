## Usage for Dev Agent

Each story file contains:
- Story ID and title
- User story and acceptance criteria
- Technical context and implementation notes
- Links to relevant architecture/PRD sections
- Definition of done checklist

**Workflow:**
1. Read the next story file
2. Implement the story
3. Mark it complete in this README
4. Move to the next story file

No need to come back to PM agent between stories - just work through them sequentially!
