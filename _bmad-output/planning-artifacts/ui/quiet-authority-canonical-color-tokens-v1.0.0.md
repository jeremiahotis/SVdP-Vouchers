# Quiet Authority  
## Shyft Ecosystem UX/UI Design System  
**Canonical Color Tokens (Light + Dark Mode)**

**Status:** Canonical  
**Applies to:** All Shyft ecosystem products (MoneyShyft, ResourceShyft, Voucher Systems, Pro Dashboards)  
**Governance Level:** Non-negotiable unless exception documented in Review Phase

---

## Quiet Authority — Core Color Tokens (Canonical · Light Mode)

### 1. Primary (Authority, Action, Trust Anchor)

**Primary / Brand Core**  
`#3F647E`  
Muted blue-slate. Reads as competent, steady, non-performative.

**Used for:**
- Primary CTAs  
- Selected states  
- Key navigation anchors  

**Primary Hover**  
`#35576D`

**Primary Active**  
`#2C4A5D`

---

### 2. Backgrounds (Calm Field, No Visual Demand)

**App Background (Default)**  
`#F6F4F1`  
Warm off-white. Avoids clinical white while remaining neutral.

**Card / Surface Background**  
`#FFFFFF`  
Pure white only for contained surfaces.

**Secondary Surface (Panels, Context Cards)**  
`#EFE9E4`  
Used for read-only context, summaries, and non-interactive containers.

---

### 3. Text (Hierarchy Without Aggression)

**Primary Text**  
`#2B2B2B`  
High legibility, not pure black.

**Secondary Text**  
`#5F6A70`  
Used for labels, metadata, helper text.

**Muted / Subtle Text**  
`#8A949A`  
Timestamps, verification notes, non-critical information only.

---

### 4. Borders & Dividers (Structure, Not Decoration)

**Standard Border**  
`#D6DAD8`

**Subtle Divider / Hairline**  
`#E4E7E5`

**Focused Border (Inputs)**  
`#3F647E`

---

### 5. Status Colors (Low-Drama Semantics)

**Success / Verified**  
`#6F8F7A`

**Warning / Attention**  
`#C6A86B`

**Error / Blocked**  
`#9A5A5A`

**Info / Neutral Status**  
`#7A8A94`

---

### 6. Interactive Neutrals (Secondary Actions)

**Secondary Button Background**  
`#E7ECEF`

**Secondary Button Text**  
`#3F647E`

**Disabled State**
- Background: `#E2E2E2`  
- Text: `#9B9B9B`

---

## Quiet Authority — Dark Mode Palette (Canonical)

### Design Intent

- Dark ≠ dramatic  
- Avoid pure black and pure white  
- Maintain the same authority center  
- Reduce eye strain  
- Preserve semantic meaning
